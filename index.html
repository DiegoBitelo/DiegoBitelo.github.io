Breve Histórico!

Criado em parceria com a Sybase, em 1988, inicialmente para complementar do Windows NT, sendo que depois passou a ser aperfeiçoado e vendido separadamente. A parceria com a Sybase terminou em 1994, e a Microsoft continuou a melhorar o programa até chegar a versões mais atuais.

Microsoft SQL Server
  • Banco de Dados Relacional(RDBMS Relational Database Management System)
  • Permite usar programação gerenciado(C# ou VB.NET) para endereçar consultas.
  • Linguagem padrão SQL (Structured Query Language)
  • Gerenciador usado Microsoft SQL Server Management Studio
Operadores Básicos
  • SELECT Recupera dados do Banco de Dados
  • INSERT,UPDATE,DELETE Insere, altera ou remove linhas de tabelas do banco de dados. São conhecidos como comandos DML (Data Manipulation Language)
  • CREATE,ALTER,DROP,RENAME,TRUNCATE Cria, altera e remove objetos do banco de dados. São conhecidos como comandos DDL (Data Definition Language)
  • COMMIT,ROLLBACK,SAVEPOINT Gerenciam as modificações realizadas pelos comandos DML. As modificações efetuadas pelos comandos DML podem ser agrupadas em transações lógicas.
  • GRANT,REVOKE Atribuem e removem direitos de acesso ao banco de dados e aos objetos a ele pertencentes. São conhecidos como comandos DCL (Data Control Language).
